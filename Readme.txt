Анализ временного ряда, который содержит цена на поездку на поездах компании Amtrak

Использованы следующие методы:
скользящее среднее, экспоненциальное сглаживание, двойное  экспоненциальное сглаживание, SARIMA.

По итогу получена модель, с ошибкой MAE = 105, что составляет 5% от средней цены билета.
